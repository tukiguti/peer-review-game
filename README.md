# 査読ゲーム (peer-review-game)

研究テーマのカードを3枚引き、その場ででっち上げた研究内容を60秒で語り、他プレイヤーが「査読が通るか」を秘密投票する対面パーティゲーム。

> 「量子コンピューティング」×「エスノグラフィ調査」×「被験者は3人まで」
> ——この3枚で、それらしい研究を語れ。過半数が Accept すれば採択。

- **形態**: Webアプリ / 1画面パスプレイ（端末1台を囲んで対面で遊ぶ・サーバ不要）
- **人数**: 3〜8人
- **1プレイ**: 5人 × 2周でおよそ25分

## 遊び方

1. **抽選** — スロットが回り、「分野」「手法」「制約」のカードが1枚ずつ決まる
2. **発表** — 発表者が60秒で研究内容を語る（制約カードの無茶ぶりは無視できない）
3. **査読** — 発表者以外が端末を回して `Accept` / `Reject` を秘密投票
4. **判定** — 過半数 Accept で採択。発表者に +2点（満場一致なら +3点）
5. 査読者も、**多数派側に投票していれば +1点**

## 開発

```bash
npm install
npm run dev      # http://localhost:5173
npm run test     # 得点計算・抽選ロジックのユニットテスト
npm run build
```
